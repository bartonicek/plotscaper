# plotscaper

<!-- badges: start -->
<!-- badges: end -->

The `plotscaper` package provides a simple, easy-to-use interface for
building interactive data visualizations geared towards exploratory data
analysis.

## Installation

You can install the development version of plotscaper from
[GitHub](https://github.com/) with:

``` r
# install.packages("devtools")
devtools::install_github("bartonicek/plotscaper")
```
